# About Siemens Interface

In the setup of the laboratory 23.N411 of the HEVS the Siemens PLC is used as interface between IO cards and CtrlX PLC.

You should not have to load the Siemens Project.
If the Siemens PLC display have the red logo of the HEVS it should be ready for the communication.
